# AFTERGLOW Report

## Concept
AFTERGLOW is a desktop Three.js light puzzle game where the CPU light field is the rule system: floor becomes solid only where direct or bounced surfel irradiance reaches the walk threshold. The project implements a CPU SurfelGI-style radiosity approximation, not DDGI or a physically exact SurfelGI renderer.

## Controls
WASD/Arrow move, mouse look in first person, E/left-click pick or place, Q cycle held color, Z undo, R reset, G GI/direct toggle, B bounce view, M peek, T third-person robot view, F1/V/N debug, C capture name, Esc title/menu.

## System Structure
The logic layer is deterministic and Node-importable: `src/gi/SurfelSolver.js`, `src/gi/sampleField.js`, `src/game/rules.js`, `src/game/levels.js`, and `src/core/math.js` own walkability, gate checks, continuous foot sampling, and validation. The visual layer reads the same surfel irradiance through `src/gi/applyGI.js`, textured materials, emissive tile states, point lights, bloom, outline, and debug surfel views. No GPU readback is used for gameplay.

## Verification
The report captures are generated by `npm run capture:report`, which starts the Vite app, plays through the game without debug teleport, toggles GI/bounce/debug views, and writes the 15 PNGs to `public/report-captures/`.

Latest local checks:
- `npm run fetch-assets`: downloaded ambientCG floor/wall/bounce textures and `RobotExpressive.glb`, exit 0.
- `npm run validate:levels`: open scarcity probe plus L1-L5 assertions passed.
- `npm run smoke:interactions`: mouse/E pick-place, ghost, undo, color, reset, GI/B, cameras, and fall/respawn passed.
- `npm run build`: level validation and Vite build passed.
- Full Playwright run: L1-L5 completed in-game, final state `GAME_COMPLETE` with `AFTERGLOW RESTORED`.

## Limitations / Fallback
The GI model is intentionally approximate and tuned for playable puzzles. `fetch-assets` writes a fallback notice and exits 0 if downloads fail; runtime then uses procedural canvas textures and the capsule robot fallback. The open-room scarcity probe currently verifies a local single-block pool of `16/196` walkable floor cells.
